# Before: Prompt-only workflow

```text
Improve this AI-native consulting proposal.
Make it clearer, more persuasive, and less generic.
Also check if it sounds credible for a B2B buyer.
```

## What is missing

- No durable goal contract.
- No explicit reader or buyer anxiety model.
- No hard-fails.
- No acceptance criteria.
- No evidence path for “credible”.
- No record of what changed and why.
- No next mutation if the proposal still fails.

The prompt may produce a better draft, but the team still has to pay Verification Tax manually.
